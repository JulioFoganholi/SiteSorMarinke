Explicações Detalhadas

HTML: É a linguagem padrão usada para criar e estruturar páginas da web. Define elementos como títulos, parágrafos, links e imagens.

CSS: Permite aplicar estilos visuais à estrutura HTML. Controla cores, fontes, tamanhos e o layout dos elementos em uma página.

JavaScript: Torna as páginas web interativas, permitindo validar formulários, criar animações, manipular elementos e muito mais.

Git: É um sistema de controle de versão usado para acompanhar as mudanças em projetos de software, facilitando o trabalho em equipe.

Docker: Plataforma que empacota aplicações e suas dependências em contêineres, garantindo que funcionem em qualquer ambiente de forma consistente.
